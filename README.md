# Nappen_Portfolio2.0
Portfolio 2.0 is the second version of a mutli-page website portfolio featuring case studies, an about me section, and provides contact information. The intent of the website is to attract recruiters for potential jobs. 

The main pages include home, about, projects, and contact. Additional pages created and linked in projects to showcase individual case studies include At Height Images, FieldHaven, Nut Tree Plaza - Holiday in the Breezeway, Nut Tree Plaza - Promotional Ads, and Siskiyou County Arts Council - Art Radio Logo.
